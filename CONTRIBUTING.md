# Contributing to Yume

Thanks for your interest in contributing to Yume!

## Ways to Contribute

### Report Bugs
- Open an [issue](https://github.com/YumeOnSol/yume/issues) with a clear description
- Include steps to reproduce the bug
- Add screenshots if applicable

### Request Features
- Open an issue with the `feature-request` label
- Describe the use case and expected behavior

### SDK Improvements
- Fork the repo
- Create a feature branch (`git checkout -b feat/my-feature`)
- Commit your changes
- Open a Pull Request

### Examples
We welcome new examples! Ideas:
- Discord bot integration
- Python wrapper
- React component
- Next.js app

## Code Style
- TypeScript for SDK code
- Use `async/await` over callbacks
- Include JSDoc comments for public methods
